## 1. Pourquoi des variables ?

Une variable sert à une chose très simple :
> **donner un nom à une valeur pour pouvoir la réutiliser**

Dans la vie courante, nous faisons cela en permanence.

Au lieu de dire :
> *la personne qui s’appelle Sahadia*

nous disons simplement :
> *Sahadia*

Le nom devient une référence.
En programmation, c’est exactement la même chose.

---

## 2. La philosophie des variables en SAHADIA

Dans SAHADIA LANGUAGE :
- une variable est une **intention claire**
- un nom doit **exprimer le sens**, pas la technique
- une variable ne cache rien

Créer une variable, ce n’est pas “déclarer un type”.
C’est **affirmer une intention**.
