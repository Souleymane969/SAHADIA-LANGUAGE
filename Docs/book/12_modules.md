# Chapitre 12  
## Modules — Organiser sans se perdre  
*Fichiers, import, séparation logique*

Quand un programme grandit,
tout ne peut pas rester
dans un seul fichier.

Les modules permettent
de séparer les idées,
de clarifier les rôles,
sans complexifier.

---

## 1. La philosophie des modules en SAHADIA

Dans SAHADIA :
- un module a une intention claire
- un fichier = une responsabilité
- l’import est explicite

Les modules ne sont pas des labyrinthes.
Ce sont des chemins.

---

## 2. Qu’est-ce qu’un module ?

Un module est simplement :
> **un fichier SAHADIA réutilisable**

Il contient :
- des fonctions
- des structures
- des constantes
